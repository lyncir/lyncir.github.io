使用Pelican写Blog

2022-02-28 10:20:00
By lyncir
Category: python
Tags: pelican

写一篇新文章

$ cat content/文件名.rst

标题
####

:date: 日期,格式 2010-10-03 10:20
:modified: 修改时间
:category: 分类,逗号分割
:tags: 标签,逗号分割
:slug: url路径

正文

使用 reStructuredText 标记语言

生成静态文件

$ make html
$ echo "本地查看"
$ make serve

上传到github

$ pelican content -o output -s pelicanconf.py
$ ghp-import output
$ git push git@github.com:lyncir/lyncir.github.io.git gh-pages:pages
