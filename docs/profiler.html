python性能分析工具

2022-03-08 11:40:00
By lyncir
Category: python
Tags: profiler

py-spy

比较好用的功能是使用 record 命令生成火焰图

$ py-spy record -o profile.svg -- python run.py

或者生成 speedscope 格式进行分析

$ py-spy record --format speedscope -o profile.speedscope.json -- python run.py

cProfile

$ cat test.py
import cProfile
pr = cProfile.Profile()
pr.enable()

code...

pr.disable()
pr.dump_stats("pipeline.prof")

$ flameprof pipeline.prof > pipeline.svg
or
$ snakeviz pipeline.prof

line_profiler

$ cat test.py
from line_profiler import LineProfiler
lp = LineProfiler()

lp_wrapper = lp(func)
lp_wrapper(**kwargs)
print(lp.print_stats())
